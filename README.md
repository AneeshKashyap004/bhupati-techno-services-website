# Bhupati Techno Services Website

Premium enterprise website for Bhupati Techno Services, built with Next.js 15, TypeScript, Tailwind CSS, Framer Motion, GSAP, React Hook Form, and Three.js.

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
npm run start
```

## Deployment

Deploy on Vercel:

1. Push this repository to GitHub, GitLab, or Bitbucket.
2. Import the project in Vercel.
3. Use the default Next.js settings.
4. Set the production domain to `www.bhupatitechno.com`.

## SEO

The site includes page metadata, Open Graph tags, JSON-LD LocalBusiness schema, semantic page structure, optimized images via Next.js remote patterns, sitemap, and robots configuration.
